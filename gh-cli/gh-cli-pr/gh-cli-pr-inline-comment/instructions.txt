GH CLI PR Inline Comments — post, edit, delete inline review comments on specific diff lines via `gh api`.

Inline comments = file+line anchored (Files Changed view). Require commit_id, path, line, side.
Required inputs: OWNER, REPO, PR_NUMBER, FILE_PATH, LINE_NUMBER, BODY.
NOT `gh pr comment` (general) — different API endpoint.

Step 1 — get commit SHA:
```bash
gh pr view {PR} --json headRefOid --jq '.headRefOid'
```

Step 2 — verify file + line in diff:
```bash
gh pr diff {PR} --name-only
gh pr diff {PR} -- {FILE_PATH}
```
`+` = RIGHT; `-` = LEFT; space = context → RIGHT.

Step 3 — deduplication check (before posting):
```bash
gh api --paginate /repos/{OWNER}/{REPO}/pulls/{PR}/comments \
  --jq '.[] | select(.path == "{FILE_PATH}" and .line == {LINE}) | {id, body, author: .user.login}'
```
If existing comment at that line — review before posting.

Step 4 — post:
```bash
gh api --method POST /repos/{OWNER}/{REPO}/pulls/{PR}/comments \
  --field body="{BODY}" \
  --field commit_id="{SHA}" \
  --field path="{PATH}" \
  --field line={LINE} \
  --field side="RIGHT"
```
Never use deprecated `position`. Use `line` (absolute line number in file).

Multi-line:
```bash
gh api --method POST /repos/{OWNER}/{REPO}/pulls/{PR}/comments \
  --field body="{BODY}" \
  --field commit_id="{SHA}" \
  --field path="{PATH}" \
  --field start_line={START} \
  --field start_side="RIGHT" \
  --field line={END} \
  --field side="RIGHT"
```

Edit:
```bash
gh api --method PATCH /repos/{OWNER}/{REPO}/pulls/comments/{ID} --field body="{BODY}"
```

Delete:
```bash
gh api --method DELETE /repos/{OWNER}/{REPO}/pulls/comments/{ID}
```

List:
```bash
gh api --paginate /repos/{OWNER}/{REPO}/pulls/{PR}/comments
```

Comments vs threads: each post creates a thread. Comment id (int) ≠ thread node_id (string).
Edit/delete via /pulls/comments/{id}. Resolve threads → gh-cli-api (resolveReviewThread GraphQL).

422 "line not part of PR" → wrong side, line not in diff, or stale SHA → re-fetch SHA + view diff.
General PR comments → gh-cli-pr-comments. Review verdicts → gh-cli-pr-review.
