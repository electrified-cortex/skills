GH CLI PR Comments — add, edit, delete general PR comments via `gh pr comment`.

Add:
```bash
gh pr comment 123 --body "text"
```

Edit — no `--edit` flag; use REST API directly: find comment ID → PATCH:
```bash
# find comment ID
gh api /repos/{owner}/{repo}/issues/{issue_number}/comments
# edit
gh api --method PATCH /repos/{owner}/{repo}/issues/comments/{comment_id} \
  --field body="updated text"
```

Delete — no `--delete` flag; use REST API directly:
```bash
gh api --method DELETE /repos/{owner}/{repo}/issues/comments/{comment_id}
```

View existing comments:
```bash
gh pr view 123 --comments
```

Resolve review threads — no `gh pr` command; use GraphQL via `gh-cli-api`:
```bash
gh api graphql -f query='
  mutation { resolveReviewThread(input: {threadId: "THREAD_ID"}) { thread { isResolved } } }'
```

Scope: `gh pr comment` only. Review-level comments (approve/request-changes verdict) → `gh-cli-prs-review`. Viewing also available in `gh-cli-prs` inspection.
